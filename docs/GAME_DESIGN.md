# Game Design

## Core Loop

Core loop is two parts:
One brother is living underground. He is expanding dungeon, digging, placing obstacles, hiring monsters and fighting adventurers. More of a dungeon keeper style game with planning, optimizing routes.
One brother is living above. He is operating a store selling dead adventurer items, earning money for dungeon expansion and town expansion for better adventurers. He is talking with villagers, completing quests.

## World Structure

One layer for outside. Hand crafted world serialized into file.
Multiple layers for underground. Part random generated, part not.
There is the crypt which connects underground with outer world, but might be more places

## Progression

Basically the more adventurers dies of, the more items from them you get. The more you sell, more you earn.
You can spend money for materials. Materials goes into town and dungeon expansion. Expanding dungeon and town attracts more and better adventurers.
Some traps, craftable objects is locked behind some progression, some behind certain resources, some maybe under some quests or so.
Progression happens with storyline as well.

## Economy

Money, Resources, Traps, Hired monsters, Relationship with town, time. All of these are some kind of resources you will have to manage in one way or another.

## Combat & Traps

Different type of traps, objects. Custom. Combar real time combat with dash, attack, guard, shoot, basic magic. Might be skill based later on, but from the beggining it will be click to attack, click to move and so on.
