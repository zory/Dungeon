# Dungeon

A 2D top-down dungeon management game built in Unity 6 (URP). Two brothers -- one living, one dead -- run parallel loops: the living brother manages a shop in the village above, while the dead brother explores procedural dungeons below. Built with a composition-based entity system and a strict Logic/Visuals/UI assembly separation.
